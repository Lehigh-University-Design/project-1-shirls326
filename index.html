History of Programming Languages on the Internet
☻

1991

Tim Berners-Lee, a computer scientist, pioneered the development of HTML (Hypertext Markup Language) as the inaugural markup language. Introduced to facilitate internet content creation, HTML structures web documents by using tags to define elements like headings and links. Berners-Lee's innovation laid the foundation for the World Wide Web, enabling the seamless exchange of information online.

1995

JavaScript and PHP were both created. JavaScript was created which enabled creation of dynamic and interactive elements within webpages (such as buttons). PHP served as a server-side scripting language.

1996

CSS was created to separate the presentation of the webpage from it's structure. Before then, everything was done in HTML. CSS allowed for more flexible and consistent designs

2009

In subsequent years, the evolution continued with the creation of Node.js, a groundbreaking development that enabled JavaScript to extend its reach to server-side operations. Serving as a versatile runtime environment, Node.js not only facilitated the execution of JavaScript on servers but also became a hub for various frameworks and libraries, further expanding the capabilities and versatility of web development.

2010 → Present

The rise of modern frameworks such as React, Angular, and Vue, begins a new revolution of web development. These tools began an era of user-friendly developer paradigms, simplifying complex tasks and enabling efficient creation of dynamic and interactive web applications. With a focus on component-based architecture, these frameworks streamline development, providing a robust foundation for the creation of responsive and feature-rich user interfaces. As they take center stage, the web scene witnesses a transformative shift towards more accessible and enjoyable development experiences.
